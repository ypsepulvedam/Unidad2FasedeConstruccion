Curso Diseño de Sitios Web

La intención de este objeto virtual de información O.V.I. es introducirlo en las temáticas del curso de Diseño de sitios web de la Universidad Nacional y a Distancia UNAD, en las cuales aprenderemos y explicaremos cómo crear repositorios por medio de la magnífica herramienta que es el sistema de control de versiones GIT. Espero sea de gran ayuda para ustedes y sirva para su crecimiento académico y personal.

OBJETIVOS

Del O.V.I.

Introducir al navegador gráfica e informativamente en la importancia del uso del sistema de control de control de versiones GIT.

OBJETIVOS

Del curso

Configurar una cuenta de usuario en la plataforma GITHUB, instalar el cliente de GIT y GITHUB para escritorio de su computador, crear un repositorio de prueba, que debe estar sincronizado con el cliente de escritorio.
